# GoldMind AI

GoldMind AI adalah AI Trading Assistant khusus XAU/USD (Gold) yang dibangun untuk membantu analisis profesional menggunakan Smart Money Concepts (SMC), ICT, Price Action, Wyckoff, Elliott Wave, Volume Analysis, serta analisis Fundamental.

## Objectives

- Analisis multi timeframe
- Institutional trading concepts
- Risk Management
- Trading Journal
- Trade Planning
- News Analysis
- XAU/USD Focus
- Exness Cent Account Optimization

## Engine

The Engine folder contains the decision-making logic used by GoldMind AI.

Unlike Knowledge, which stores concepts and reference material, Engine defines how the AI evaluates information, scores setups, applies rules, and reaches a trading decision.
