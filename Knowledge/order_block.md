# Order Block

## Bullish Order Block

Last bearish candle before bullish displacement.

## Bearish Order Block

Last bullish candle before bearish displacement.

## Valid Order Block

Must:

- Cause BOS.
- Show displacement.
- Align with HTF.
- Not be fully mitigated.

## Invalid Order Block

- Counter-trend
- No BOS
- Weak displacement
- Already mitigated
