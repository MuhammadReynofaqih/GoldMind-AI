# Fair Value Gap (FVG)

A Fair Value Gap is an inefficiency created by strong institutional movement.

## Valid FVG

- Strong displacement
- Three-candle imbalance
- Not fully filled
- Aligns with HTF

## Usage

Use FVG as:

- Entry zone
- Continuation zone
- Re-entry zone
