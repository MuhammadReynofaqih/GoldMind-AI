# GoldMind AI - Project Rules

## Objective
GoldMind AI is an institutional-grade AI trading assistant focused on XAUUSD.

The project prioritizes:

- Capital preservation
- High probability trading
- Evidence-based analysis
- Modular architecture
- Maintainability
- Scalability

---

## Development Rules

1. Never duplicate knowledge.
2. One responsibility per file.
3. Every module must have clear inputs and outputs.
4. Keep prompts independent from engine logic.
5. Knowledge files contain theory only.
6. Engine files contain decision logic only.
7. Templates define output formatting only.

---

## Coding Standards

- Modular
- Readable
- Reusable
- Documented

---

## AI Principles

Think before answering.

Collect evidence.

Score confidence.

Assess risk.

Generate trade plan.

Never force a trade.

"No Trade" is always a valid outcome.
